# Hotel Management System Mockup

A comprehensive hotel management system with TV interface for guests and staff dashboard for hotel operations.

[Add your GitHub Pages link here after deployment]

## 📋 Features

### Guest TV Interface
- Room service ordering (food & beverages)
- Hotel services booking (housekeeping, spa, etc.)
- Order history and tracking
- Shopping cart functionality
- Room information display

### Staff Dashboard
- Multi-department login (Front Desk, Kitchen, Management, etc.)
- Real-time order management
- Inventory tracking and alerts
- Room status monitoring
- Guest management
- Reporting and analytics

## 🛠️ Technologies Used

- HTML5, CSS3, JavaScript (ES6+)
- Local Storage for data persistence
- Responsive design
- Font Awesome icons

## 🎯 How to Use

### For Guests (TV Interface)
1. Open `index.html` in a web browser
2. Navigate using the menu to order food or services
3. Add items to cart and place orders

### For Staff (Dashboard)
1. Open `front-desk.html` in a web browser
2. Login with demo credentials:
   - **Front Desk**: user: `frontdesk`, pass: `123`
   - **Kitchen**: user: `kitchen`, pass: `123`
   - **Management**: user: `admin`, pass: `123`
3. Manage orders, inventory, and rooms from the dashboard
